Practice Homework

Variable Cake, Oh my!

Make a program that uses variables to print out your favorite cake!

The program should ask for your favorite cake and store it into a variable, then concatinate it with the string " is my favorite cake" before printing it to the screen.

If you are not comfortable with variables yet, try to make a program that will "stick together" a string of your favorite cake and the string " is my favorite cake".

Helpfull Stuff

  • The command "input()" is used to get user input
  • Use + to concatinate strings
  • Don't forget the "quotes" around your strings!
