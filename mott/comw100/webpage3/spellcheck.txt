In web page design html tags are used to "mark up" so that a computer AND a human
can both read its contents. Html offers many options for marking text (there are literaly
boundless options with special div and span tags). To use an html tag place the text that
you want to mark for the comutper inside of an opening bracket pair (&lt;&gt;) and a closing braket
pair preceded by a / (&lt;/&gt;), the name of the tag that you want to use goes in the &lt;&gt; .
Example: &lt;p&gt;this text is marked as a paragraph&lt;/p&gt;. There are loads of tags that you can use
to give your document special meaning to the computer! For example, the html pre tag is used to create text 
that the browser renders with its white space in-tact, normaly the browser ignores white space altogether!
You can use the pre tag for poems where the formating of the text matters and the whitespace will stay with the text!
In addition to the pre tag html offers TONS more tags to learn and use, but if none of them are to your liking 
you can allways specify a div or span tag to make your own! div and span tags are general purpos tags in the
browser that have very little special meaning. You can give them meaning yourself by using 
the id or class attrobutes. To use an attribute place the attribute name followed by an equal sign
after the name of the tag in the opening braket pair. 
Example: &lt;div class="potato"&gt; potato related content here &lt;/div&gt;
make sure that you surround any text to the left of equal signs with quotes!
