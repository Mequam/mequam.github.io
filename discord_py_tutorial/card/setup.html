How To Make A Bot

Just like users, all discord bots need an account. We need to create one before we can use our bot!

After that we need to add our bot to a server, it can only talk to us if we decide to listen.

Saving The Token

Just like you have a password to log into accounts your bot has a secret word to tell discord that it can login. You need to save this token after you make the bot

Remember:

NEVER SHARE YOUR TOKEN
Setting Up discord.py

We need to install discord.py before we can use it

pip is a python package manager with all sorts of goodies we can use it to install discord.py!

pip install -U discord.py

if pip is not set in the path variable then it can be found at this link

C:/Users/{Your Name}/AppData/Local/Programs/Python/Python310/Scripts
